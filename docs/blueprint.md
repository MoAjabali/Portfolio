# **App Name**: Stardust Portfolio

## Core Features:

- About Me Section: Display personal information and a brief introduction.
- Education Summary: Showcase academic qualifications, including institutions attended and degrees obtained.
- Skills Showcase: Highlight technical and soft skills with visual progress bars.
- Project Gallery: Present a selection of completed projects with descriptions and links to live demos or repositories.
- Contact Section: Provide a contact form and contact information, facilitating direct communication.

## Style Guidelines:

- Background color: Black (#000000) for a sophisticated dark mode interface.
- Primary color: White (#FFFFFF) for clear contrast and readability.
- Accent color: Subtle Blue (#ADD8E6) as an accent color to highlight interactive elements sparingly.
- Font pairing: 'Space Grotesk' (sans-serif) for headers, giving a computerized, space-age feeling; and 'Inter' (sans-serif) for body text.
- Use line-style icons to create a modern feel. Icons can relate to study and work.
- Single page layout with smooth scrolling to each section.
- Subtle parallax scrolling effect for background images, creating an illusion of depth.
